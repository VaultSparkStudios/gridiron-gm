# Project Brief

## Identity

- Name: Gridiron GM
- Slug: `gridiron-gm`
- Type: Browser-based game (football GM simulation)
- Status: Live — actively developed
- Owner: VaultSpark Studios

## Why this exists

- Problem: No lightweight, browser-native NFL-style franchise simulator exists that runs without a backend or install
- Audience: Casual football fans and sim enthusiasts who want a quick but deep franchise management experience
- Studio importance: Flagship sports game; demonstrates the studio's ability to ship polished single-file browser games

## Success

- What success looks like: A player can run a full multi-season franchise — draft, trade, simulate, win championships — entirely in the browser with no install, no login, no ads
- What this project is not: Not a real-money game, not multiplayer, not an official NFL product

## Scope

- In scope: Roster management, NFL Draft (7 rounds), free agency, game simulation, playoffs, live play-by-play, scouting, trades, multi-season progression
- Out of scope: Real NFL player names/likenesses, multiplayer, persistent cloud saves, monetization
